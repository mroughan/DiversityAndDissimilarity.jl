# Manual Download Queue

This file lists reference materials that could not be downloaded directly by
`curl` from an open, stable source during the reference audit. Each item has an
anchor matching its BibTeX key so `notes/references.bib` can still point to a
local traceable file.

Where an HTML metadata page or failed-download page was saved locally, it is
listed as a local aid. The requested library copy should be stored in
`notes/references/` using the suggested filename.
