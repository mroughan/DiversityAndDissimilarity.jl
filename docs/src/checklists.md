# Availability Checklists

These checklists compare `DiversityAndDissimilarity.jl` with commonly used
biodiversity and community ecology packages: Diversity.jl, vegan, iNEXT,
scikit-bio, EcoPy, Microbiome.jl, SciPy, SpadeR, and entropart.

Two checklists are provided:

- [**Diversity Index Checklist**](index-checklist.md) — alpha-diversity,
  entropy, richness estimation, evenness, and single-assemblage indices.
- [**Similarity And Dissimilarity Checklist**](similarity-checklist.md) —
  incidence, abundance, and probability-based pairwise indices.

Each row uses the following key:

| Symbol | Meaning |
|---|---|
| `[x]` | Available directly as a named or documented function/method |
| `[~]` | Available indirectly, derivable from another output, or available with a different convention |
| `[ ]` | Not documented as available in the checked source |

Convention differences (such as the Simpson naming issue) are noted in the
rightmost column of each table.
