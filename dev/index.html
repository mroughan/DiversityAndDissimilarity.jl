DiversityAndDissimilarity.jl

DiversityAndDissimilarity provides ecology-style diversity indices for species abundance data, observation vectors, and pairwise assemblage comparisons.

The package is built around small dispatchable index types and generic operations:

  • entropy for Shannon, Renyi, and Tsallis entropy values.
  • diversity for alpha diversity and effective diversity.
  • effective_diversity for effective species numbers.
  • similarity, dissimilarity, and distance for pairwise comparisons.
  • counts and proportions for preparing abundance data.
  • alpha_diversity for a compact exploratory alpha-diversity summary.
  • index_metadata, index_bounds, descriptor helpers such as is_metric and is_similarity, reference_cases, and diversity_audit for convention-aware and validation-oriented workflows.

If you are coming from R, start with the vegan migration guide. If you are choosing between packages, the diversity and similarity/dissimilarity checklists summarize where DiversityAndDissimilarity.jl overlaps with common ecology and biodiversity tools.

Supported Julia Versions

DiversityAndDissimilarity supports Julia 1.10 and newer 1.x releases.

Quality Checks

The standard test suite runs across the supported Julia compatibility matrix. A separate GitHub Actions quality workflow runs Aqua.jl and JET.jl checks from the test/quality environment.

Index Types

Alpha diversity:

  • Richness
  • Shannon, which defaults to base=2 for entropy in bits
  • Renyi
  • Tsallis
  • Simpson
  • GiniSimpson
  • InverseSimpson
  • Hill
  • Chao1
  • ACE
  • SampleCoverage
  • PielouEvenness
  • FisherAlpha

Pairwise comparisons:

  • Jaccard
  • SorensenDice
  • BrayCurtis
